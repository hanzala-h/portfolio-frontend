## Getting Started

**Clone the repository**

```bash
git clone https://github.com/hanzala-h/portfolio-frontend.git
cd portfolio-frontend
```

**Install dependencies**

```bash
npm install
```

**Start development server**

```bash
npm run dev
```

**Run linting**

```bash
npm run lint
```

**Format code**

```bash
npm run format
```
